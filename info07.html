Guía de instalación WRF model parte II

Guía instalacion por Lainer

Mayo 29, 2025

En esta segunda parte de la guía de instalación del modelo WRF( Weather Research and Forecasting Model), se detallan los pasos necesarios después de preparar el entorno del sistema, instalar las bibliotecas requeridas y realizar pruebas de compatibilidad antes de compilar el modelo. El WRF es un modelo numérico ampliamente utilizado en meteorología y climatología para simular y predecir fenómenos atmosféricos a diferentes escalas.

La instalación del WRF es un proceso que requiere una cuidadosa preparación del entorno de software y hardware. El modelo WRF es altamente configurable y puede adaptarse a diversas necesidades de investigación y pronóstico meteorológico. Su diseño modular permite a los usuarios seleccionar diferentes configuraciones y opciones según sus requisitos específicos, lo que lo convierte en una herramienta versátil para la simulación atmosférica.

Ahora pasaremos a la instalación del WRF y WPS (WRF Preprocessing System), que es un sistema de preprocesamiento de datos meteorológicos utilizado para preparar las condiciones iniciales y de contorno necesarias para ejecutar el modelo WRF. El WPS incluye herramientas para interpolar datos de entrada, generar campos de geografía estática y crear archivos de entrada necesarios para las simulaciones del WRF.

Instalación del WRF y WPS

En esta sección, se describen los pasos necesarios para instalar el modelo WRF y el sistema de preprocesamiento WPS. Estos pasos son fundamentales para garantizar que el modelo funcione correctamente y pueda utilizarse para realizar simulaciones precisas de fenómenos atmosféricos.

Antes de comenzar la instalación, es importante asegurarse de que se hayan completado las pruebas del entorno del sistema y de que se hayan instalado las bibliotecas necesarias Descrita en la parte I de este turorial. Estas pruebas y configuraciones previas son esenciales para garantizar que el modelo WRF se compile y ejecute sin problemas.

Instalación de WRF

Para instalar el modelo WRF, vamos a seguir una serie de pasos que incluyen la compilación del código fuente, la configuración de las opciones de compilación y la generación de los archivos ejecutables necesarios. Estos pasos son cruciales para preparar el modelo para su uso en simulaciones meteorológicas.

Pasos para la instalación del WRF

  • Descargar el Modelo WRF de la paguina web de github clonando el repositorio o descarganda el archivo .tar.gz de la version de WRF de interés
    • git clone --recursive-submodules https://github.com/wrf-model/WRF
    • Link de descarga a WRF v4.5.1
  • Descomprimir el fiche .tar.gz descargado en el directorio de trabajo y moverse al directorio WRF.
    • tar -xvf v4.5.1.tar.gz
    • cd WRF
  • Ejecutar el script de configuración para seleccionar las opciones de compilación y el tipo de ejecución que se desea utilizar. Esto se puede hacer ejecutando el siguiente comando:
    • ./configure
    • Dentro del script de configuración se debe escoger el set de compiladores en función de las capacidades de su PC, servidor o cluster
  • Correr el script de compilación
    • ./compile em_real
    • Este comando compilará el modelo WRF con las opciones seleccionadas en el paso anterior. Dependiendo de la configuración y la potencia de la máquina, este proceso puede tardar un tiempo considerable.

Puede verificar que la instalación se haya realizado correctamente ejecutando el siguiente comando: ./run_wrf.exe -h o verificar que se hayan generado los archivos ejecutables en el directorio de trabajo con el siguiente comando: ls -lh main/*exe

Instalación de WPS

El WPS es un sistema de preprocesamiento de datos meteorológicos que se utiliza para preparar las condiciones iniciales y de contorno necesarias para ejecutar el modelo WRF. La instalación del WPS sigue un proceso similar al del WRF, pero con algunas diferencias específicas.

Para instalar el WPS, se deben seguir los siguientes pasos:

  • Descargar el WPS de la página web de github clonando el repositorio o descargando el archivo .tar.gz de la versión de WPS de interés
    • git clone https://github.com/wrf-models/WPS
    • Link de descarga del WPS 4.5
  • Descomprimir el fichero .tar.gz descargado en el directorio de trabajo y moverse al directorio WPS.
    • tar -xvf v4.5.tar.gz
    • cd WPS
  • Ejecutar el script de configuración para seleccionar las opciones de compilación y el tipo de ejecución que se desea utilizar. Esto se puede hacer ejecutando el siguiente comando:
    • ./configure
    • Dentro del script de configuración se debe escoger el set de compiladores en función de las capacidades de su PC, servidor o cluster
  • Correr el script de compilación
    • ./compile
    • Este comando compilará el WPS con las opciones seleccionadas en el paso anterior. Dependiendo de la configuración y la potencia de la máquina, este proceso puede tardar un tiempo considerable.
