¿Cómo grabar tu pantalla en Wayland con Gnome?

#gnome #wayland #screenrecorder #pipewire

Mayo 17, 2024

Este es un tutorial para poder realizar screen recording o grabar pantalla, tanto video como audio. Todo esto, con un programa compatible con Wayland y para Gnome Desktop. Lo he probado con buenos resultados en diferentes configuraciones de hardware. No lo he probado en KDE, ni en Cinnamon, ni en otros entornos de escritorios similares. Supongo que pueda funcionarte. Se que es compatible con Wayland, y pipiware. Me imgino que no debe se compatible con pulseaudio pero hay un montón de screenrecording compatibles con X11 y pulseaudio, ejemplo: Kazam, GNOME’s Screen Recorder, etc.

El sotware a utilizar para grabar pantalla es Wayfarer. Un software en desarrollo que se encuentra en github. El escritor de este post no está relacionado con el desarrollador del software, ni es colaborador del desarrollador principal. Lo he usado como usuario con buenos resultados. Después de esta breve introdución comencemos con los pasos a seguir para instalar el software:

Primer Paso: Clonar el repositorio de github (Esto puede variar con el tiempo) Revise si el link esta activo o disponible.

git clone https://github.com/stronnag/wayfarer.git

Segundo Paso: Muevase al fichero wayfarer

cd wayfarer

Tercer Paso: Instalé la librerías y los programas necasrios

sudo apt install libgtk-4-dev valac libgstreamer1.0-dev libpulse-dev libayatana-appindicator3-dev gir1.2-ayatanaappindicator3-0.1 libgstreamer-plugins-base1.0-dev
sudo apt install cmake meson ninja-build

Cuarto Paso Compilar Wayfarer

meson build
ninja -C build

Quinto paso: Instalación de Wayfarer

sudo ninja -C build install

Con esto debe tener ya Wayfarer instalado en sus sistema GNU/Linux y podrá grabar sus escritorios siun muchos problema. El programa es bastante intuitivo pero si tiene algun poblema en entender ¿cómo usarlo como usurio? vaya a la página del desarrollador de Wayfarer para mas detalle.

Si desea desintalar, auxíliese de la herramienta synaptic.
